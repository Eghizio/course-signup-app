# Extracurriculum
An application that enables __students__ to *sign up* for extracurricular activities, and __teachers__ to *browse*, *manage* and *export* the data.

## Installation
>`npm install`
## Development mode
>`npm run-script dev`
## Build
>`npm run-script build`
## Run
>`npm start`
